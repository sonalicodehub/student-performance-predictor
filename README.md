# 🎓 Student Performance Prediction

A Machine Learning project that predicts a student's final exam score based on various academic and personal factors such as attendance, scores in subject, and previous performance and visaualize the performance graphically.

## 📌 Overview

This project demonstrates an end-to-end Machine Learning workflow:
- Generate a realistic student dataset
- Train a regression model
- Predict students' final marks
- Evaluate model performance
- Visualize the performance

## 🚀 Features

- 📊 Automatically generates a student dataset
- 🤖 Trains a Machine Learning regression model
- 🎯 Predicts final exam scores
- 💾 Saves the trained model for future predictions
- 📈 Easy to modify and extend

## 🛠️ Tech Stack

- Python
- NumPy
- Pandas
- Scikit-learn
- Joblib
- Matplotlib

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-sonalicodehub/student-performance-prediction.git
```

Install the required packages:

```bash
pip install pandas numpy scikit-learn joblib
```

## ▶️ How to Run

Generate the dataset:

```bash
python generate_dataset.py
```

Train the model:

```bash
python train_model.py
```

Predict student performance:

```bash
python predict.py
```
```bash
python visualize.py
```
